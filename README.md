# Group4TAFlavorshare - Recipe App

## What is this?
This is a fun recipe-sharing app! Share your favorite recipes, browse others, and sign up with a @gmail.com email. It's like a digital cookbook for everyone.

## How to Play (Use It)
1. Go to the live site: https://denbelloga.github.io/Group4TAFlavorshare/
2. Sign in with any @gmail.com email (like test@gmail.com) and any password.
3. Browse recipes by clicking cards—they open in a popup!
4. Upload your own recipe by filling out the form.
5. Try it on your phone—it works there too!

## Cool Features
- Sign in/up with Gmail emails only.
- Browse recipe cards with pictures.
- Upload recipes with title, description, and ingredients.
- Works on phones and computers.
- Easy to use with buttons and popups.

## Made With
- HTML (for the pages)
- CSS (for the pretty looks)
- JavaScript (for the magic actions)

Have fun cooking and sharing! If you find bugs, tell us.
